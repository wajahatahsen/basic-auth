## Description
A basic JWT authentication project in NestJS. 

## Startup Instructions
### env file
CREATE a .env file and add following properties in it
```
NODE_ENV=local
DB_HOST=localhost
DB_PORT=27017
DB_NAME=auth
APP_PORT=9000
```

### Install dependencies:
`npm i`

### Start the project:
`npm start`


## Testing Instructions
### Use following command to run the unit tests
`npm test`
